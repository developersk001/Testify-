## ✨ Overview

Testify JEE is a modern, offline-first web application for generating
personalized JEE Main and JEE Advanced mock tests from a structured
question bank. It provides timed exams, rich analytics, revision tools,
and a clean, premium interface.

## 🌟 Features

-   🎯 Custom test generation
-   📚 Previous-year question architecture
-   ⏱️ Auto-submit timer
-   📊 Advanced analytics dashboard
-   📈 Subject/chapter/topic insights
-   📌 Bookmarks & Mistake Book
-   🔁 Revision Mode
-   💾 Local storage (IndexedDB + LocalStorage)
-   🌙 Dark & Light themes
-   📱 Fully responsive
-   ⚡ PWA-ready

## 🚀 Getting Started

``` bash
git clone https://github.com/your-username/testify-jee.git
cd testify-jee
npm install
npm run dev
```

Visit `http://localhost:3000`.
